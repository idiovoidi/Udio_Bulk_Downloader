## Troubleshooting

### Can't find the extension icon?
- Click the puzzle icon in Chrome toolbar
- Find "Udio Library Mapper"
- Click the pin to keep it visible

### No folders/songs found?
- Make sure you're logged in
- Wait for the page to fully load
- Refresh and try again

### Export not working?
- Check Chrome download settings
- Try the other export format
