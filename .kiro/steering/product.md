# Product Overview

Udio Downloader is a Python utility for downloading songs and folder structures from the Udio music platform before its shutdown. The tool authenticates with Udio, maps the user's library structure (folders and songs), and downloads all content while preserving the original organization.

## Key Features

- Automated authentication and session management with Udio platform
- Library structure mapping (folders and songs hierarchy)
- Bulk download with concurrent processing
- Resume capability for interrupted downloads
- Metadata preservation
- Chrome DevTools Protocol integration for browser automation

## Target Use Case

Users need to backup their entire Udio library before platform shutdown, maintaining folder organization and song metadata.
